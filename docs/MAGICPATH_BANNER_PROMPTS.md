## üö® Troubleshooting

### If MagicPath doesn't generate correctly:

**Problem: Text is too small**
Solution: Explicitly state "make text highly readable, minimum 14px equivalent"

**Problem: Colors are off**
Solution: Use exact hex codes in quotes: "background color #0a0a0a exactly"

**Problem: Layout is misaligned**
Solution: Provide pixel-perfect positioning: "position at x=420px, y=50px exactly"

**Problem: Wrong dimensions**
Solution: Start prompt with "IMPORTANT: Canvas must be exactly 1584x396 pixels"

**Problem: Too much decoration**
Solution: Add "minimal, clean design, no unnecessary elements"

## üí° Pro Tips for MagicPath AI

1. **Be specific with dimensions** - Always state exact pixel sizes
2. **Use hex codes** - Don't say "teal," say "#14b8a6"
3. **Reference fonts by name** - "Inter Bold" not just "bold"
4. **Iterate in steps** - Generate base, then refine
5. **Save versions** - Keep working files for future edits
